xcodebuild

Exposes warnings, errors and test results. It requires a JSON generated using xcpretty-json-formatter, to be passed as an argument for it to work.

Created by Valerio Mazzeo.

Reference

Attributes

Allows you to specify a build title to prefix all the reported messages.

build_titleString

Allows you to specify an xcodebuild JSON file location to parse.

json_file

Allows you to specify whether default messages should be posted when parsing

post_messagesBool

Methods

Parses and exposes eventual warnings.

parse_warnings warning_count

Parses and expose eventual errors.

parse_errors error_count

Parses and exposes eventual test failures.

parse_tests test_failures

Prints "Perfect build üëçüèª" if everything is ok after parsing.

perfect_build is_perfect_build

Examples

Ensure the project compiles without warnings, errors and all tests are executed correctly

xcodebuild.parse_warnings
xcodebuild.parse_errors
xcodebuild.parse_tests
xcodebuild.perfect_build
