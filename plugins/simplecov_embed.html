simplecov

Report your Ruby app test suite code coverage.

You can use simplecov to gather code coverage data and a json formatter so this plugin can parse it.

Created by Marcelo Fabri.

Reference

Methods

Parse a JSON code coverage file and report that information as a message in Danger.

report

Examples

Report code coverage

simplecov.report('coverage/coverage.json')
